# Excel Data Analysis Projects

This repository contains Excel-based data analysis projects focused on data cleaning, analysis, and visualization.

The projects cover core analytical skills such as:
- Data cleaning and structuring
- Excel formulas (VLOOKUP/XLOOKUP, IF, SUMIFS, INDEX-MATCH)
- Pivot tables and pivot charts
- Basic dashboards and visual insights

These projects were developed as part of the *Data Analysis Essentials Using Excel* course and further organized and documented for portfolio purposes.

## Tools & Skills
- Microsoft Excel
- Data Cleaning
- Data Analysis
- Pivot Tables
- Data Visualization

## Project Structure
Each folder represents a separate project and includes:
- Raw or cleaned datasets
- Excel analysis files
- Visualizations and dashboards (where applicable)
