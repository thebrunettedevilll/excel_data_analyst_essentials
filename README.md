# Excel Data Analysis Projects

This repository contains a structured collection of Excel-based data analysis projects developed to demonstrate core and advanced analytical skills.

The projects cover the full Excel data analysis workflow — from formulas and statistics to pivot tables and data visualization — and are organized for clear portfolio presentation.

## Repository Structure

### 
Core Excel formulas and financial calculations, including:
- Mathematical and logical functions
- Conditional calculations
- Error handling and formula auditing
- Financial metrics (NPV, IRR, CAGR, loan amortization)

### 
Statistical and exploratory data analysis concepts, including:
- Descriptive statistics (mean, median, mode)
- Percentiles and distributions
- Variance and standard deviation
- Correlation analysis
- Trend identification

### 
Advanced data retrieval and logic techniques, including:
- VLOOKUP, HLOOKUP
- INDEX and MATCH (single and multi-condition)
- DSUM and conditional lookups
- Named ranges
- Nested IF logic and rankings

### 
End-to-end pivot table workflows, including:
- Data preparation for pivoting
- Pivot table creation and modification
- Calculated fields
- Grouping, sorting, filtering, and slicers
- Updating pivots with new data

### 
Data visualization and reporting using Excel charts, including:
- Line, bar, column, and area charts
- Scatter and bubble charts
- Combo and waterfall charts
- Gantt and bullet charts
- Best practices for insight-driven visual storytelling

## Skills Demonstrated
- Excel formulas and functions
- Data cleaning and preparation
- Statistical analysis and EDA
- Pivot tables and data summarization
- Data visualization and reporting
- Financial analysis fundamentals

## Tools
- Microsoft Excel

These projects were developed as part of a structured Excel data analysis curriculum and organized to showcase practical, job-relevant analytical skills.
