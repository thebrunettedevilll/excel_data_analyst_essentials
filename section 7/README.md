This folder contains Excel workbooks demonstrating a wide range of chart types used for data visualization and analytical reporting.

Each file focuses on selecting, building, and formatting the appropriate chart to effectively communicate insights from data.

## Included Chart Types

- **Area Chart**  
  Visualizing cumulative trends over time.

- **Bar Chart**  
  Comparing values across categories.

- **Bubble Chart**  
  Representing relationships between multiple numerical variables.

- **Bullet Chart**  
  Comparing performance against targets.

- **Clustered Column Chart**  
  Side-by-side comparison of multiple data series.

- **Combo Chart**  
  Combining different chart types for enhanced insight.

- **Gantt Chart**  
  Visualizing project timelines and task dependencies.

- **Line Chart**  
  Tracking trends and changes over time.

- **Pie Chart**  
  Displaying proportions within a whole.

- **Scatter Chart**  
  Analyzing relationships and correlations between variables.

- **Stacked Column Chart**  
  Comparing totals while showing category contributions.

- **Waterfall Chart**  
  Visualizing incremental changes contributing to a final value.
