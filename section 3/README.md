This folder contains a collection of Excel workbooks designed to practice and demonstrate core data analysis and financial calculation concepts.

Each file focuses on a specific Excel topic, covering formulas, functions, and analytical techniques commonly used in data analysis and business contexts.

## Included Topics & Files

- **Anchoring and Naming Cells**  
  Absolute and relative cell references, named ranges, and best practices for structured formulas.

- **Array Functions**  
  Use of array formulas to perform calculations across multiple values.

- **Basic Mathematical Operations**  
  Fundamental calculations including sum, subtraction, multiplication, and division.

- **Compound Annual Growth Rate (CAGR)**  
  Calculation and interpretation of compound growth over time.

- **Conditional Arithmetic**  
  Conditional calculations using logical criteria.

- **Formula Auditing**  
  Tracing precedents and dependents to validate and debug formulas.

- **Internal Rate of Return (IRR)**  
  Evaluation of investment profitability using IRR calculations.

- **IS Functions**  
  Error checking and data validation using IS functions.

- **Loan Amortization**  
  Construction of loan amortization schedules.

- **Logical Operators (IF, AND, OR)**  
  Logical decision-making within Excel formulas.

- **Minimum, Maximum & Averages**  
  Use of MIN, MAX, MINIFS, MAXIFS, AVERAGE, and AVERAGEIFS functions.

- **Net Present Value (NPV)**  
  Discounted cash flow analysis using NPV.

- **Rounding Numbers**  
  Controlling numerical precision using rounding functions.

- **SUM & SUMIF**  
  Aggregation and conditional summation techniques.
