# Excel Lookup, Logic & Data Validation – Practice Files

This folder contains Excel workbooks focused on advanced lookup techniques, logical formulas, and data validation methods used in data analysis workflows.

The files demonstrate how to efficiently retrieve, filter, validate, and rank data within Excel.

## Included Topics & Files

- **Avoid Errors with IF and ISNUMBER**  
  Error handling and validation using logical and IS functions.

- **Create Dropdowns for Filtering**  
  Data validation techniques using dropdown lists to control inputs.

- **Create Unique Lists**  
  Extraction of unique values for analysis and reporting.

- **Fast Lookup with INDEX and MATCH**  
  Efficient and flexible data retrieval using INDEX–MATCH.

- **Lookup with HLOOKUP**  
  Horizontal lookup operations.

- **Lookup with VLOOKUP**  
  Vertical lookup operations for data matching.

- **Multi-Condition Lookup with DSUM**  
  Conditional aggregation across multiple criteria.

- **Multi-Condition Lookup with INDEX and MATCH**  
  Advanced lookups using multiple conditions.

- **Naming Arrays (VLOOKUP, HLOOKUP)**  
  Use of named ranges to improve formula readability and maintainability.

- **Nested IF Statements**  
  Complex logical decision structures.

- **Rankings with SMALL and LARGE**  
  Ranking and ordering values within datasets.

## Skills Demonstrated
- Advanced Excel lookups
- Logical and conditional formulas
- Data validation and error handling
- Efficient data retrieval
- Ranking and filtering techniques

## Tools
- Microsoft Excel
